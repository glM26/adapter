    • Defined in SASViyaApiClient.ts:1150

    Moves a Viya folder to a new location. The folder may be renamed at the same time.

    Parameters

    • sourceFolder: string

      The full path to the source folder to be moved (eg /Public/example/myFolder)

    • targetParentFolder: string

      The parent folder to which the sourceFolder will be moved (eg /Public/newDestination). To move a folder, a user has to have write permissions in targetParentFolder. If moving to recycle bin, 'targetParentFolder' will be a uri.

    • targetFolderName: string

      The name of the "moved" folder. If left blank, the original folder name will be used (eg myFolder in /Public/newDestination/myFolder for the example above). Optional field.
